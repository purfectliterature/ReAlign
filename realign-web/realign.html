How to install ReAlign on Chrome

  1. Download this file: realign-1.1.0.zip.
  2. Extract the file to somewhere.
  3. Run Chrome and navigate to chrome://extensions or click the three-dotted menu > More tools > Extensions.
  4. On the top right corner, enable Developer Mode.
  5. Click on Load unpacked and select the realign-1.1.0 folder extracted just now.
    or, you can just drag-and-drop the folder onto the Extensions page.
  6. You should be able to see the little R on the left of the omnibar.
  7. Try navigating to any Coursemology mission and see if the misalignment has been fixed.
